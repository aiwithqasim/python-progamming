## Project Overview

This repository provides a structured learning path for Python programming, covering:
- Python fundamentals and syntax
- Data structures and control flow
- Object-oriented programming (OOP)
- File handling and data processing
- A practical desktop application project

Whether you're new to programming or looking to strengthen your Python skills, this repository offers progressive learning materials from beginner to intermediate level.

## Prerequisites

Before getting started, ensure you have:

- **Python 3.x** installed ([Download here](https://www.python.org/downloads/))
- **Visual Studio Code** with Python extension ([Download here](https://code.visualstudio.com/))
- **Jupyter** extension for VS Code (for viewing/editing `.ipynb` files)

### Verify Installation

Open a terminal/command prompt and run:

```bash
python --version
```

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aiwithqasim/python-progamming.git
   cd python-progamming
   ```

2. **Open in VS Code:**
   ```bash
   code .
   ```

3. **Install recommended extensions:**
   - Python
   - Jupyter
   - Jupyter Notebook Renderers

4. **Start learning:**
   - Begin with `week1/class1.ipynb` for Python basics
   - Progress through weekly materials in order
   - Complete assignments to reinforce learning

## Project: Class Management System

A practical desktop application demonstrating real-world Python development.

### Features

- Add/delete students
- Add/delete courses
- Enroll students in courses
- Manage enrollments
- Persistent data storage with SQLite

### Technologies Used

- **GUI Framework:** Tkinter (built-in with Python)
- **Database:** SQLite3 (built-in with Python)
- **No external dependencies required**

### Running the Project

Navigate to the project directory and run:

```bash
cd project
python app.py
```

For more details, see [project/README.md](./project/README.md)
