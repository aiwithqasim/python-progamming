# Class Management System (Python + Tkinter)

A simple desktop project for managing:
- Students
- Courses
- Enrollments

It uses:
- `tkinter` for the user interface
- `sqlite3` for data storage

## Features

- Add and delete students
- Add and delete courses
- Enroll students in courses
- Delete enrollments
- Data is saved in `class_management.db`

## Run

From the `project` folder:

```bash
python app.py
```

## Notes

- No external package installation is required.
- Works with standard Python (3.x) that includes `tkinter`.
